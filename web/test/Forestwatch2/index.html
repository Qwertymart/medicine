Инструкция по использованию

Мы подробно рассмотрим, как использовать сервис ForestWatch Приморье для распознавания участков

  • 01

    Откройте вкладку Форма для загрузки

  • 02

    Добавьте один или несколько файлов формата: JPG, PNG

  • 03

    Нажмите кнопку отправить и наблюдайте результат в форме Ваш результат

  • 04

    Цвета выделенных участков:
    • Гарь
    • Лес
    • Вырубка
    • Пашня

О нас

Мы используем передовые алгоритмы машинного обучения и анализ больших данных для обработки спутниковых снимков Приморского края. Наш сервис обеспечивает высокую точность распознавания лесных участков, вырубок, гарей и пашни, предоставляя актуальную информацию для мониторинга лесных ресурсов.
